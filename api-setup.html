はじめての方向け：OpenAI API の準備

このページでは、CAサポートAPPで使う OpenAI API キーを、口座開設のイメージでゼロから用意する流れを説明します。画面の表示は OpenAI 側の更新で少しずつ変わることがありますが、大まかな順序は同じです。

⚠️ 課金・残高について（必ずお読みください）

  • 利用には料金がかかります。API を呼び出すたびに、OpenAI アカウントの残高（クレジット）から従量課金で引かれます。
  • 初めてのチャージは 5 ドル（USD）程度を推奨します。お試し利用なら十分なことが多く、必要になったらあとから追加で購入（チャージ）できます。
  • このアプリが自動で OpenAI に課金したり、カードから勝手に引き落としたりすることはありません。チャージや支払いは、必ず OpenAI の公式サイト（Billing）で行います。
  • OpenAI 側の「自動リロード（残高が減ったら自動でチャージ）」などの設定は、使わない・オフにすることを推奨します。意図しない請求を防ぐため、都度、自分で残高を確認してから追加購入する運用にしてください。
  1. 1

    OpenAI のアカウントを作る

    メールアドレスで登録し、必要なら電話番号の確認などを済ませます。

    1. ブラウザで https://platform.openai.com/signup を開きます。
    2. 画面の案内にしたがってアカウントを作成し、ログインできる状態にします。
  2. 2

    支払い方法を登録し、残高をチャージする

    API を使うには課金用の設定が必要です。5 USD 前後から始めるのが無難です。

    1. Billing（請求・残高）の画面を開きます。
    2. クレジットカードなどの支払い方法を追加します（表示は英語のことが多いです）。
    3. Add to credit balance や Buy credits などのボタンから、少額（推奨：5 ドル）を購入して残高を作ります。
    4. 残高が少なくなったら、同じ画面から再度、自分で追加購入してください。アプリ側から残高を増やす操作はできません。
  3. 3

    API キーを発行する

    キーはパスワードと同じです。他人に見せたり、メールで送ったりしないでください。

    1. API keys のページを開きます。
    2. Create new secret key（新しいシークレットキー）を押し、名前を付けて作成します。
    3. 表示された文字列（sk- で始まる）をすぐにコピーします。この画面を閉じると再表示できないことが多いので、メモアプリなどに一時保存しても構いません（共有厳禁）。
  4. 4

    CAサポートAPP に貼り付ける

    キーはブラウザ内にだけ保存され、当社のサーバーには送られません。

    1. アプリ（index.html）を開きます。
    2. 右上の 「設定」を押し、コピーした API キーを入力して 「保存する」を押します。

    本アプリは GPT-4o などのモデルを利用します。OpenAI アカウント側でモデルや利用地域の制限がある場合は、公式ドキュメントもあわせてご確認ください。
